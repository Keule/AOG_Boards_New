# Hardwaremodule

## Navigation

- Das System MUSS zwei UM980 verwenden.
- Jeder UM980 MUSS einen eigenen UART verwenden.
- Baudrate MUSS 921600 sein.
- Heading MUSS aus zwei Empfängern berechnet werden.

## Steering

- Das System MUSS BNO085 verwenden.
- Das System MUSS ADS1118 verwenden.
- ADS1118 MUSS 128 SPS verwenden.
- Das System MUSS DRV8263H verwenden.
- Steuerung MUSS über PWM/DIR erfolgen.
- Safety MUSS vorhanden sein.
- Watchdog MUSS vorhanden sein.
- Fehler MUSS Ausgänge hochohmig setzen.
